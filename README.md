# Задание по Flutter: Композиция виджетов и работа с API

## Описание

Это задание предназначено для закрепления знаний по основам Flutter:
- Композиция стандартных виджетов (Column, Expanded/Flexible, SizedBox, Text, Image)
- Работа с асинхронными данными (FutureBuilder)
- Запросы к API (http пакет)
- Структурирование кода и вынесение компонентов в отдельные виджеты

## Установка

1. Убедитесь, что у вас установлен Flutter SDK
2. Установите зависимости:
   ```bash
   flutter pub get
   ```

## Запуск

```bash
flutter run
```

## Структура задания

Все задание находится в файле `lib/main.dart`. В комментариях подробно описаны:
- Основное задание
- Дополнительные задания
- Требования к структуре кода
- Подсказки по использованию виджетов и API

## API

Используется API [wttr.in](https://wttr.in/) для получения данных о погоде.

Формат запроса: `https://wttr.in/{город}?format=j1`

Пример: `https://wttr.in/Moscow?format=j1`

## Основные требования

1. Реализовать функцию `fetchWeather()` для получения данных из API
2. Использовать композицию виджетов: Column, Expanded/Flexible, SizedBox, Text, Image, FutureBuilder
3. Отобразить данные о погоде: город, иконку, температуру, описание
4. Добавить кнопку обновления данных
5. **Важно**: Не допускать большую вложенность виджетов. Выносить компоненты в отдельные виджеты (StatelessWidget или отдельные методы)

## Дополнительные задания

### Дополнительное задание 1: Scrollable list
Создайте прокручиваемый список с данными о погоде для трех городов:
- Москва
- Париж
- Лондон

Используйте `ListView` или `ListView.builder` для реализации.

### Дополнительное задание 2: Theme
Используйте `Theme` для настройки темы приложения:
- Настройте цвета и стили текста через `ThemeData`
- Добавьте переключатель между светлой и темной темой
- Используйте `Theme.of(context)` для доступа к теме в виджетах

## Подсказки

- Используйте `Expanded` для виджетов, которые должны занимать доступное пространство
- `Flexible` позволяет виджету занимать часть доступного пространства
- `SizedBox` полезен для создания отступов и фиксированных размеров
- `FutureBuilder` автоматически перестраивает виджет при изменении состояния Future
- Не забывайте обрабатывать ошибки и состояния загрузки
- Выносите сложные виджеты в отдельные классы для улучшения читаемости кода
